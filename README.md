# Search Engine Project

## Описание проекта
Реализация поискового движка для корпоративного портала.

## Стек технологий
- C++17
- Google Test
- JSON for Modern C++

## Структура проекта
- `src/` – исходные коды
- `tests/` – модульные тесты
- `resources/` – файлы для поиска
- `CMakeLists.txt` – сборка проекта

## Сборка и запуск
```bash
mkdir build
cd build
cmake ..
cmake --build .
./search_engine
